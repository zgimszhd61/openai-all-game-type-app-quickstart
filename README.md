### 原问题
什么样的游戏不需要通过版号审批？

### 本质问题
哪些类型的游戏在中国不需要通过版号审批？

### 答案
在中国，一般来说，以下类型的游戏不需要通过版号审批：

1. **单机游戏**：如果是完全离线运行的单机游戏，且不涉及到任何网络功能（例如联网对战、社交功能、在线商店等），通常不需要版号。
2. **试玩版、测试版**：尚未正式上线的试玩版或测试版游戏，通常不需要版号，但需要标明是测试版并仅限于小范围测试。
3. **教育类游戏**：一些主要用于教育目的的游戏，如果其内容完全符合教育类产品的定义，并且不涉及商业运营，可能不需要版号。
4. **小程序小游戏**：在微信、支付宝等平台上的一些小程序小游戏，尤其是非盈利性质的，可能不需要版号，但这类游戏在发布前需要与平台确认相关政策。

需要注意的是，这些规则可能会随着政策变化而有所调整，开发者在发布游戏前应确保了解最新的法规和要求。
